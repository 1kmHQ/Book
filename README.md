## How to Read This

Read it slowly. These ideas are meant to settle, not to impress.

If something feels obvious, good. The best principles are obvious in hindsight.

If something feels uncomfortable, pause. That discomfort often points to where conventional thinking has failed rural India.

---

## A Note on Language

This book avoids jargon. It speaks in plain terms about real problems faced by real people in real places.

The farmer in Dharapuram and the driver in Hosur should be able to read this and say: *"Yes, this is my life. This is my problem. This makes sense."*
